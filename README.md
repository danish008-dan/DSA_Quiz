High Frequency Interview Problems
==================================

Problems Covered
----------------
1. LRU Cache
2. Median of Two Sorted Arrays

LRU Cache
---------
Data Structures Used:
- HashMap
- Doubly Linked List

Time: O(1)
Space: O(n)

Median of Two Sorted Arrays
---------------------------
Technique:
- Binary Search on smaller array
- Partition logic

Time: O(log(min(n, m)))
Space: O(1)

These are considered:
- Hard level problems
- Common in FAANG interviews
